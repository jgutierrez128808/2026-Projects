# Daisy Seed Projects

Audio projects for the Electro-Smith Daisy Seed.

## Setup

- [libDaisy](https://github.com/electro-smith/libDaisy) — hardware abstraction layer
- [DaisySP](https://github.com/electro-smith/DaisySP) — DSP library (oscillators, filters, effects)
- [Getting Started Guide](https://electro-smith.github.io/libDaisy/)
