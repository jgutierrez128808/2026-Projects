# 2026 Projects

Synthesizer and embedded audio projects using Teensy 4.1, Daisy Seed, and Arduino.

## Structure

```
2026-Projects/
├── Teensy/       # Projects for Teensy 4.1
├── DaisySeed/    # Projects for Daisy Seed
└── Arduino/      # Projects for Arduino
```

## Platforms

- **Teensy 4.1** — Used with the [PJRC Audio Library](https://www.pjrc.com/teensy/td_libs_Audio.html)
- **Daisy Seed** — Used with [libDaisy](https://github.com/electro-smith/libDaisy) and [DaisySP](https://github.com/electro-smith/DaisySP)
- **Arduino** — General prototyping and control
